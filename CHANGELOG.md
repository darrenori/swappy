# Changelog

Built for a school module. Rough order, not a formal release history.

- Core app: auth, encrypted JWT sessions, storefront, cart, checkout with OTP, reviews
- Back office: product manager, store manager, employee manager, tasks, attendance, logs
- Security library (includes/security): session hardening, headers, CSP, rate limiting,
  honeypot, upload validation, audit log, maintenance mode
- Helpers (includes/helpers): formatting, text, pagination, http/flash
- Validators (includes/validate) and prepared-statement db helpers (includes/db)
- Error handling and JSON response helpers
- Standalone tests and a lint script
